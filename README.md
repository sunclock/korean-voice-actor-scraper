# korean-voice-actor-scraper

## 기능

한국 성우 정보를 불러옵니다.
국내 오디오드라마 정보를 불러옵니다.
오디오 드라마와 한국 성우 정보를 매칭합니다.

## 사용 방법

1. `chmod +x execute.sh`
2. `./execute.sh`
3. 출력된 Failure List 결과를 복사하여 actors_exception.py의 failures 변수에 저장.
4. `python3 actors_exception.py`
